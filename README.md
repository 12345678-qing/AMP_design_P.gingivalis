# AI-Guided Antimicrobial Peptide Design for Oral Pathogens

## 项目简介
本仓库用于口腔致病菌靶向抗菌肽的机器学习研究，包含数据集、模型训练代码、可解释性分析（SHAP）代码及实验结果。

适用于论文初稿、审稿复现、科研存档。

## 目录结构
- data/        数据集（抗菌肽序列、标签等）
- code/        模型训练、特征工程、SHAP分析代码
- results/     实验结果、图表、输出文件

## 使用方法
代码基于Python，主要依赖：
- pandas
- numpy
- scikit-learn
- shap
- matplotlib
